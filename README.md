# 🎩 Lupin Desktop Pet

> *"Se mi prendi ti ridò il tuo desktop esattamente com'era!"*

Un desktop pet in stile **Arsène Lupin** che ruba le icone del desktop Windows e ti sfida a catturarlo.

![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey.svg)

---

## 🚀 Quick Start

### 🎨 Versione 2D - Leggera e Classica
```bash
cd pet/2d-version
pip install -r requirements.txt
python main.py
```

### 🎮 Versione 3D - Completa e Avanzata
```bash
cd pet/3d-version
pip install -r requirements.txt
python main.py
```
