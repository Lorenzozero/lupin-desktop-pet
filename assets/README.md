# assets

Metti qui la demo animata con nome **`demo.gif`** — è il file referenziato
nell'header del README principale.

Suggerimento per registrarla: usa [ScreenToGif](https://www.screentogif.com/)
(gratis, Windows), cattura ~6-10 secondi mentre Lupin ruba un'icona e scappa, ed
esporta come `demo.gif` (max ~5 MB così carica veloce su GitHub).
