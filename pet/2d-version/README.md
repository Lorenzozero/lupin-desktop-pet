# 🎩 Lupin Desktop Pet - 2D Version

Versione classica 2D con **PyQt5**, sprite animati, particelle avanzate e effetti visivi.

## 🚀 Installazione

```bash
cd pet/2d-version
pip install -r requirements.txt
python main.py
```

## ✨ Features

- Sistema particelle avanzato (sparkle, smoke, cuori)
- Sound effects con pygame
- Combo system con moltiplicatori
- Icon ghosts (icone che volano nel sacco)
- Cursor attention (occhi seguono mouse)
- Emotion system
- Motion trail durante corsa
- Screen shake e flash effects
- 3 personalità (aggressive, playful, sneaky)
